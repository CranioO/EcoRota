# EcoRota
# 🌍 Mapa Interativo de Coleta de Resíduos (João Pessoa - PB)

Este projeto foi desenvolvido como parte das Atividades de Extensão do meu curso de Ciência da Computação. O objetivo é facilitar o descarte correto de resíduos, conectando cidadãos a pontos de coleta específicos.

## 🚀 Tecnologias Utilizadas
* **HTML5/CSS3:** Estruturação e estilização da interface.
* **JavaScript (ES6):** Lógica de filtros e manipulação de dados.
* **Leaflet.js:** Biblioteca para mapas interativos.
* **Supabase:** Backend-as-a-Service para armazenamento de dados em tempo real.

## 🎯 Impacto Social
Alinhado aos **ODS 11 e 12 da ONU**, o sistema ajuda na redução do descarte irregular de lixo eletrônico e plásticos, promovendo a economia circular na região de João Pessoa.

## 🛠️ Como testar
Basta abrir o arquivo `index.html` em qualquer navegador.
